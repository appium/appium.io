Filozofia Appium

Ideą Appium jest by testowanie aplikacji natywnych nie wymagało dodawania specjalnych SDK czy też rekompilacji Twojej aplikacji. Powinieneś również mieć możliwość użycia preferowanych przez siebie narzędzi, jęzków i praktyk. Appium to w pełni otwarty projekt, stworzony tak, by zachęcić społeczność do udziału w jego rozbudowie.

Użyj ulubionych narzędzi

Celem Appium jest automatyzacja każdej aplikacji mobilnej z dowolnego języka programowania przy użyciu dowolnego framework'a testującego, z pełnym dostępu do API back-end'u oraz daz danych z kodu testującego. Pisz testy przy użyciu swoich ulubionych narzędzi używając dowolnego z powyższych języków .. i nie tylko (możesz użyć Selenium WebDriver API i jego implementacji dla różnych języków).

Wymagania

iOS

Mac OSX 10.7+
XCode 4.5+ z Command Line Tools

Android

Mac OSX 10.7+ lub Windows 7+ lub Linux
Android SDK ≥ 16 (SDK < 16 w trybie Selendroid)

Dziecinnie prosta instalacja. Spróbuj teraz!

Zainstaluj node.js (zawiera npm, packet manager dla node.js). 

> sudo npm install appium -g
> npm install wd
> appium &
> node your-appium-test.js

Zobacz: przykładowe testy Appium.
