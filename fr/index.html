Appium

Automatisation pour les Apps.

Appium est un système de test d'automatisation à <<open source>> ou code source ouvert pour l'utilisation avec des applications mobiles natives ou hybrides.
Il guide les applications iOS et Android en utilisant le WebDriver JSON wire protocol.

La philosophie Appium

Appium est construit sur l'idée que tester des applications natives ne devrai pas requérir <<SDK>> ou un kit de développement logiciel ou la reconstruction de votre application. Et que vous devriez pouvoir utiliser vos pratiques de test, système et outils préférrés. Appium est un projet open source et a pris des décisions de conception et d'outil pour encourager une communauté dynamique contributive..

Utilisez vos outils préférés

Appium vise à automatiser n'importe quel app depuis n'importe quel langage et n'importe quel système de test avec un accès complet au API back-end et aux bases de données à partir du code test. Ecrivez des tests avec vos outils de développement préférés dans tous les langages montrés ci-dessus et certainement encore plus (avec le API WebDriver Selenium et des bibliothèques clientes spécifiques à la langue)

Exigences

iOS

Mac OSX 10.7+
XCode 4.5+ avec outils de ligne de commande

Android

Mac OSX 10.7+ ou Windows 7+ ou Linux
Android SDK ≥ 16 (SDK < 16 en mode Selendroid)

Procédé de configuration facile, lancez un test maintenant.

Installez node.js (npm, le gestionnaire de paquets node.js inclus) 

> npm install -g appium
> npm install wd
> appium &
> node votre-appium-test.js
