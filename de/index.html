Appium

Testautomatisierung für Apps

Appium ist ein Open Source Testautomatisierungswerkzeug für native und hybride mobile Anwendungen (Apps).
Das Werkzeug testet iOS und Android Apps mit Hilfe des WebDriver JSON Wire Protocol.

Appium Philosopie

Appium basiert auf der Grundidee keinerlei Modifikationen (z.B. SDKs, Re-Kompilierung) an nativen Apps vornehmen zu müssen, um automatisierte Tests gegen diese Apps ausführen zu können. All Das unter der Zuhilfenahme etablierter Testmethodiken, -Frameworks und -Tools. Appium wurde auf den Fundamenten von Open Source Prinzipien entworfen, um Beiträge und Verbesserungen aus der Entwicklergemeinschaft zu fördern.

Testen in gewohnter Umgebung

Appiums Ziel besteht darin jegliche mobile App mit Hilfe jeglicher Programmiersprache/Testframeworks und mit vollem Zugriff auf Back-End Schnittstellen und Datenbanken zu automatisieren. Programmiere Tests in deiner favorisierten Sprache unter der Zuhilfenahme der oben aufgelisteten Technologien. Jede Programmiersprache, die über Selenium WebDriver API Client-Bibliotheken verfügt ist Appium-kompatibel.

Systemanforderungen

> Mac OSX 10.6 +
> XCode
> Apple Developer Werkzeuge
(iPhone Simulator, CLI-Werkzeuge)
> Android SDK API >= 17

Kinderleichte Installation. Jetzt ausprobieren!

Installiere node.js (beinhaltet npm, den node.js Paketmanager). 

> sudo npm install appium -g
> npm install wd
> appium &
> node your-appium-test.js
