在设备上激活指定的 app

示范用法

  • Java
  • Python
  • Javascript
  • Ruby
  • C#
  • PHP 
driver.activateApp('com.apple.Preferences');
driver.activateApp('io.appium.android.apis');


driver.activate_app('com.apple.Preferences')
driver.activate_app('io.appium.android.apis')


// webdriver.io 例子
driver.activateApp(null, 'com.apple.Preferences')
driver.activateApp('io.appium.android.apis')

// wd 例子
// iOS 端仅在使用 XCUITEST 框架前提下支持 `mobile: queryAppState` 命令


# ruby_lib 例子
activate_app('com.apple.Preferences')
activate_app('io.appium.android.apis')

# ruby_lib_core 例子
@driver.activate_app('com.apple.Preferences')
@driver.activate_app('io.appium.android.apis')


driver.activateApp('com.apple.Preferences');
driver.activateApp('io.appium.android.apis');


// iOS 端仅在使用 XCUITEST 框架前提下支持 `mobile: queryAppState` 命令

说明

通过 XCUITest 框架，iOS 端测试同样可以使用 mobile: activateApp 方法。详情请查看文档。

HTTP API 规范

Endpoint

POST /session/:session_id/appium/device/activate_app

URL 参数

字段名 说明
session_id 通过命令去路由的session id

JSON 参数

字段名 类型 说明
bundleId 或 appId string iOS 端使用 BundleId， Andorid 端使用包名(package name)。

响应（Response）

无
