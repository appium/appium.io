Appium

Automatización para Aplicaciones Moviles

Appium es un framework de código abierto de testeo automatizado para el uso con aplicaciones nativ as e híbridas. Conduce a aplicaciones iOS y Android usando WebDriver y el protocolo JSON .

Filosofía de Appium

Appium se basa en la idea de que las pruebas de aplicaciones nativas no debería necesitar de un SDK o recompilar la aplicación. Y que usted debería ser capaz de utilizar con sus prácticas preferidas de pruebas, frameworks y herramientas. Appium fue diseñado sobre la base de los principios de código abierto con el fin de alentar las contribuciones y mejoras de la comunidad de desarrolladores.

Usa tus herramientas favoritas

Appium pretende automatizar cualquier aplicación móvil usando o no un marco de programación / prueba y el pleno acceso a interfaces del lado del servidor y bases de datos. Escribe pruebas con tus herramientas favoritas en los lenguajes mencionados anteriormente y mas probablemente ( con el Selenium WebDirver API y algunas librerias en lenguajes especificos).

Requisitos

iOS

Mac OSX 10.7+
XCode 4.5+ w/ Command Line Tools

Android

Mac OSX 10.7+ or Windows 7+ or Linux
Android SDK ≥ 16 (SDK < 16 en modo Selendroid)

Fácil proceso de configuración, ejecuta un test ahora.

Instala node.js (incluye npm, el manejador de paquetes de node.js). 

> sudo npm install -g appium
> npm install wd
> appium &
> node your-appium-test.js
